# Основные функциональные блоки NeuroHub

## 1. Главная страница и онбординг
- Лендинг с ценностным предложением для фрилансеров и заказчиков
- Кейсы успешных проектов по категориям (текст, изображения, видео, аудио)
- FAQ
- Двухролевая регистрация (фрилансер/заказчик) с выбором специализации
- Верификация email
- Персональные рекомендации после регистрации

## 2. Личный кабинет фрилансера

### Профиль и портфолио:
- Система навыков через теги (GPT-4, Midjourney, DALL-E и др.)
- Портфолио с примерами работ
- **Интеграция Hugging Face Spaces** для "живых" демонстраций моделей
- Рейтинг и отзывы от заказчиков
- Статистика доходов и активности

### Поиск проектов:
- Лента с фильтрами (категория, бюджет, срок, уровень)
- Умные уведомления о релевантных проектах
- Система откликов с шаблонами
- История откликов со статусами

### Управление заказами (Kanban):
- Колонки: Входящие → В работе → На проверке → Завершённые
- Встроенный мессенджер с файлами и push-уведомлениями
- Трекер времени для почасовой оплаты
- Система сдачи работы и правок (бесплатные/платные)

## 3. Личный кабинет заказчика

### Создание проекта:
- Пошаговый wizard
- Категории: текст, изображения, видео, аудио, программирование, консультации
- Бюджет (фиксированный/почасовой)
- Техническое задание
- Шаблоны проектов для повторного использования

### Поиск исполнителей:
- Каталог фрилансеров с фильтрами
- Просмотр портфолио и тест "живых" демо (Hugging Face)
- Система приглашений фрилансеров в проект

### Управление проектами:
- Просмотр и сортировка откликов
- Выбор исполнителя
- Контроль этапов и прогресса
- Приём работы и система отзывов

## 4. Технические особенности
- **Backend:** Next.js монолит с feature-based структурой, PostgreSQL 16 + Knex.js
- **Frontend:** Next.js 16 App Router + React 19, Material UI 7.3+, Effector для state
- **API:** RESTful API Routes с OpenAPI/Swagger документацией
- **Аутентификация:** Better Auth с cookie-based сессиями, OAuth (GitHub, Google)
- **Безопасность:** HTTPS, bcrypt, CSP, CSRF-токены, Zod валидация
- **Интеграция:** Hugging Face Spaces через iframe

## 5. Ключевые метрики MVP
- Конверсия регистрации
- Активность пользователей (DAU/WAU)
- Количество проектов и откликов
- Конверсия в успешные заказы
- Время до первого заказа для фрилансеров
